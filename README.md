## 🛠 Instalar
- Debes tener `pnpm` instalado

- Descargar el repositorio
    ```
    git clone https://github.com/Angelito91/angelito91.github.io.git
    ```
- Instalar las dependencias
    ```
    pnpm install
    ```
- Ejecuta
    ```
    pnpm run dev
    ```

### 📌 Tecnologías
- Astro 🚀
- JavaScript ⚡
- Tailwindcss ⭐
